מערכת לניהול מערך סיוע בדרכים (Roadside Assistance Management System)
פתרון ארגוני מקצה לקצה (Full-Stack) שנועד לייעל ולנהל קריאות סיוע ואירועי דרכים. המערכת מנהלת רישום מתנדבים, יצירת פניות וקריאות שירות, ותעדוף אוטומטי של המשימות על פי רמות דחיפות.

🛠️ טכנולוגיות (Tech Stack)
צד שרת (Backend)
סביבת הרצה: Node.js

                                                                                                                                               Express.js :Framework 

בסיס נתונים: MongoDB

                                                                                                                                                      ODM: Mongoose

צד לקוח ( Frontend - בתהליך פיתוח אקטיבי)
ספרייה ראשית: React

                                                                                                                                                         Redux :State

קליינט HTTP: Axios

עיצוב וממשק: Bootstrap

📐 ארכיטקטורה ומאפיינים מרכזיים
ארכיטקטורה מודולרית: נבנה תוך הפרדה מוחלטת בין שכבות (Separation of Concerns), תוך ניתוק הלוגיקה העסקית משכבת הגישה לבסיס הנתונים, במטרה להבטיח תחזוקה קלה ויכולת הרחבה בעתיד.

תעדוף חכם: מימוש לוגיקה מותאמת אישית לתעדוף דינמי של קריאות סיוע נכנסות על פי מדדי דחיפות קבועים מראש.

מודל נתונים חזק: שימוש בסכמות של Mongoose לצורך ולידציה וניהול קשרים מורכבים בין מתנדבים, פונים ויומני סיוע.

עיצוב API בתצורת REST המקובלת: נתיבים (Endpoints) נקיים ומאובטחים המותאמים לביצועים אופטימליים ואינטגרציה חלקה עם אפליקציית הלקוח העתידית.
